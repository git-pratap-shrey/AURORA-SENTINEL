WEIGHTED SCORING CONFIGURATION
================================================================================

Formula: Final Score = (0.3 × ML Score) + (0.7 × AI Score)

This is ALWAYS applied (no conditional logic):
- 30% weight to ML detection (fast physical patterns)
- 70% weight to AI intelligence (accurate context)

Benefits:
✅ Consistent scoring across all scenarios
✅ Reduces false positives (boxing vs real fights)
✅ Balances speed and accuracy
✅ No MAX fallback needed

Example:
  ML: 85/100 (high movement)
  AI: 50/100 (real fight detected)
  Final: 0.3×85 + 0.7×50 = 60.5/100
  Alert: YES (> 60 threshold)

HOW TO USE
================================================================================

1. Quick Start:
   python test_integrated_system.py

2. Start Backend:
   python -m uvicorn backend.api.main:app --reload --port 8000

3. Start AI Layer (optional):
   cd ai-intelligence-layer
   python server_local.py

4. Enable GPU (5-10x faster):
   - Install CUDA Toolkit
   - System will automatically use GPU

5. Add Gemini API (best accuracy):
   - Get API key: https://makersuite.google.com/app/apikey
   - Add to .env: GEMINI_API_KEY=your_key

6. Install Ollama (fallback):
   - Download: https://ollama.com/download
   - Run: ollama pull llava:7b
